pandas-data-cleaning-project
This project demonstrates practical data cleaning workflows in Python using the pandas library. It covers essential steps for transforming raw, messy datasets into clean, structured tables suitable for analysis.

Features
Load, inspect, and profile real-world tabular data

Handle missing values and inconsistent entries

Normalize column formats and text case

Split and reformat address data

Drop irrelevant columns




Requirements
Python 3.7+
pandas




Install dependencies with:
pip install pandas




Learning Goals
Understand common data issues in tabular data

Practice cleaning strategies for real analysis tasks

Learn pandas functions for efficient preprocessing
